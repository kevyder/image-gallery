# Image Gallery

2016/12/28

## It Use:

* Sinatra 2.0.8.1

* SQLite3 1.4.2

* Bootstrap 3.3.7

* CarrierWave 2.1

* HAML 5.1.2

* Sequel 5.32

* rmagick 4.1.2

## Usage

Clone this repository executing:

    $ git clone https://github.com/kevyder/image_gallery.git

And execute:

    $ cd image_gallery

To install the project gems execute:.

    $ bundle install

To run the project execute:

    $ rackup -p <PORT>
